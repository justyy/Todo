接口汇总页

注册

接口地址：/regist

提交参数

提交方式：POST

输入参数 是否必要 描述
name 必要 用户名。用户名只能是字母数字下划线和小数点！
p1 必要 密码。密码长度在6~11之间，只能包含字符、数字和下划线
nick 必要 真实姓名。只能填写中文姓名
phone 必要 手机电话必须11位。
mail 必要 公司邮箱
team 必要 1:交互组，2：设计组，3：前端组，5：多app组
sendMail 必要 是否需要邮件提醒功能。0：不需要；1：需要

返回数据

返回参数 描述
status 0为失败，1为成功
message 当status为0的时候才有的错误提示

登录

接口地址：/login

提交参数

提交方式：POST

输入参数 是否必要 描述
username 必要 用户名。
password 必要 密码。
remark 不必要 是否需要记录登录状态。值为1则保存登录cookie，为0则不保存。

返回数据

返回参数 描述
status 0为失败，1为成功
message 当status为0的时候才有的错误提示

登出

接口地址：/logout

返回数据

返回参数 描述
status 0为失败，1为成功
message 当status为0的时候才有的错误提示
